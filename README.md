# my-coursera-project-# User Stories for Smart Clinic System

## Admin
- As an Admin, I want to add or remove doctors from the system.
- As an Admin, I want to manage patient appointments.

## Doctor
- As a Doctor, I want to view my daily consultation schedule.
- As a Doctor, I want to update patient records after a visit.

## Patient
- As a Patient, I want to find a specialist (e.g., Cardiologist).
- As a Patient, I want to book an appointment with a specific doctor.
